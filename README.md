# SSH Host Manager

Simple CLI to manage your ssh shortcuts

## Install

```bash
npm install -g ssh-host-manager
```

## Requirements

* Unix system
* NodeJs
* OpenSSH with min version 7.3p1

## Usage

```bash
// Add a new entry with name test
shm add test root@example.com

// Connect to the new entry
ssh test
```
